# Shiny Python App

This is a simple interactive web application built using **Shiny for Python**.  
The app demonstrates basic interactivity with a slider input and dynamic text output.

---

## Features
- Slider input
- Dynamic text output

---

## How to run locally

```bash
pip install -r requirements.txt
shiny run app.py
``
